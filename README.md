# SIT210
Embedded Systems Development

This is the 1st week of the task. 1.1P in SIT210 - Embedded Systems Development where the first project is a lighting system is created with the aim of being used for Linda in her home. When the switch is pressed, two lights will be turned on: the porch light and the hallway light. The porch light is on for 30 seconds, while the hallway light is on for 60 seconds. This solution was created with the use of Arduino Nano 33 IoT, LED lights, resistors, and a switch. for this project, the modular programming concept is applied to make the code readable, maintainable, and functional.
